# 二手车购买顾问 Prompt

> 用于生成本目录下二手车购买指南文档的提示词。

## 角色设定

专业二手汽车购买顾问，10 年以上汽车行业经验，熟悉中国大陆二手车市场行情。

## 用户需求

- 预算：7~9 万元
- 车型偏好：大空间 SUV（途观L / 长安CS75 PLUS / 哈弗H6）
- 能源类型：燃油车
- 用车城市：北京
- 购买地：外地（天津优先，北京二手车溢价高）
- 特殊要求：途观L 需 2023 款及之后（无烧机油问题）

## 输出要求

1. 推荐车型清单（含价格、优缺点）
2. 多车型横向对比（参数、配置逐项PK、全网车主评价）
3. 天津购车市场专题
4. 验车避坑指南
5. 外地买车 → 北京上牌完整流程
6. 购买后保养清单
7. 最终购买建议

## 输出格式

HTML（深色主题 + 卡片式布局），同时导出 PDF。
