# 本地开源模型搭建方案（环境隔离）

> 硬件：RTX 5070Ti（12GB 显存）+ 32GB 内存 + 2TB SSD

## 三层隔离架构

```
1. Conda 虚拟环境（隔离 Python 版本和依赖）
   ├── llm        → 文本大模型
   ├── comfyui    → 文生图 + 图生视频
   └── dev        → 日常开发/实验

2. 模型文件统一管理（避免重复下载）
   └── ~/ai-models/
       ├── llm/
       ├── diffusion/
       ├── video/
       ├── lora/
       └── controlnet/

3. Docker（可选，更强隔离）
   └── 用于部署对外服务时
```

---

## 第一步：基础环境

### 1.1 确认 NVIDIA 驱动 + CUDA

```bash
nvidia-smi
# 确认 Driver Version: 560+，CUDA Version: 12.x
```

### 1.2 安装 Miniconda

```bash
# https://docs.conda.io/en/latest/miniconda.html
conda --version
```

### 1.3 创建统一模型目录

```bash
mkdir -p ~/ai-models/{llm,diffusion,video,controlnet,lora}
```

---

## 第二步：LLM 文本模型环境

### 方案 A：Ollama（推荐新手）

```bash
# 安装后直接运行
ollama run qwen2.5:14b-instruct-q4_K_M    # 中文首选
ollama run deepseek-r1:14b                  # 推理/代码
ollama run llama3.1:8b-instruct-q8_0        # 英文

# 改模型存储位置（Windows）
set OLLAMA_MODELS=D:\ai-models\llm
```

### 方案 B：Conda + transformers（更灵活）

```bash
conda create -n llm python=3.11 -y
conda activate llm
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
pip install transformers accelerate bitsandbytes llama-cpp-python

# 验证 GPU
python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_name(0))"
```

---

## 第三步：文生图 + 图生视频环境

### 3.1 ComfyUI 环境

```bash
conda create -n comfyui python=3.11 -y
conda activate comfyui
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

cd ~/ai-projects
git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI
pip install -r requirements.txt

# 启动：python main.py → 浏览器打开 http://127.0.0.1:8188
```

### 3.2 模型软链接（统一管理）

```bash
# Windows（管理员 cmd）
mklink /D "ComfyUI\models\checkpoints" "D:\ai-models\diffusion"
mklink /D "ComfyUI\models\loras" "D:\ai-models\lora"
mklink /D "ComfyUI\models\controlnet" "D:\ai-models\controlnet"

# Linux/Mac
ln -s ~/ai-models/diffusion ~/ai-projects/ComfyUI/models/checkpoints
ln -s ~/ai-models/lora ~/ai-projects/ComfyUI/models/loras
ln -s ~/ai-models/controlnet ~/ai-projects/ComfyUI/models/controlnet
```

### 3.3 下载模型

- **FLUX.1-dev**（文生图首选）：https://huggingface.co/black-forest-labs/FLUX.1-dev
- **SDXL**（备选）：https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0
- **Wan2.1-1.3B**（图生视频首选）：https://huggingface.co/Wan-AI/Wan2.1-T2V-1.3B

### 3.4 安装 ComfyUI Manager

```bash
cd ~/ai-projects/ComfyUI/custom_nodes
git clone https://github.com/ltdrdata/ComfyUI-Manager.git
# 重启 ComfyUI 后可一键安装插件和模型
```

## 第四步：开发实验环境

```bash
conda create -n dev python=3.11 -y
conda activate dev
pip install jupyter notebook openai anthropic langchain requests httpx
```

## 环境切换速查

```bash
conda activate llm       # 跑文本模型
conda activate comfyui   # 跑文生图/视频
conda activate dev       # 写代码/实验
conda env list           # 查看所有环境
conda env remove -n 环境名  # 删除环境（不影响其他）
```


## 训练顺序建议

| 阶段 | 内容 | 预计时间 |
|------|------|---------|
| Week 1 | 装好基础环境 + Ollama 跑通 LLM 对话 | 1-2 天 |
| Week 1 | 装好 ComfyUI + SDXL 跑通第一张图 | 1-2 天 |
| Week 2 | 学习 SDXL 提示词技巧 + LoRA 使用 | 3-4 天 |
| Week 3 | 升级到 FLUX.1-dev，对比效果差异 | 2-3 天 |
| Week 3 | 安装 ControlNet，学习构图控制 | 2-3 天 |
| Week 4 | 装 Wan2.1 / CogVideoX，跑通图生视频 | 2-3 天 |
| Week 4 | 串联完整工作流：文字→图片→视频 | 2-3 天 |
| Week 5+ | 进阶：LoRA 微调自己的风格模型 | 持续 |
