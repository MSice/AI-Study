# 智能钓鱼助手 Web 应用开发方案

## 技术栈

- **框架**: Next.js 15 (App Router)
- **UI**: React 19 + Tailwind CSS 4 + shadcn/ui
- **图表/可视化**: Recharts（时间轴）+ 自定义 SVG（水层剖面图）
- **地图**: 高德地图 JS API 2.0（Web 版，免费额度足够）
- **状态管理**: Zustand（轻量）
- **LLM 调用**: DeepSeek V3.2 API（通过 Next.js API Routes 代理）
- **气象数据**: 和风天气 API
- **数据持久化**: 初期用 localStorage，后续可接 Supabase/PlanetScale

## UI 设计风格：户外运动深色主题

- **主色调**: 深蓝灰底 (`#0f1923`) + 森林绿强调色 (`#2d8a4e`) + 琥珀色高亮 (`#d4a843`)
- **卡片**: 半透明磨砂玻璃效果 (`backdrop-blur`)，圆角 12px，微妙边框
- **字体**: 数据用等宽字体，标题用粗体无衬线
- **图标**: 自然/户外风格线条图标
- **动效**: 水波纹加载动画，卡片入场渐显

## 分阶段实施

### Phase 1: 纯 UI 界面（Mock 数据，不接任何 API）

先用硬编码的 Mock 数据渲染所有页面，确认视觉效果和交互流程。

**P1-1: 项目初始化 + 全局样式**

- `npx create-next-app` 初始化项目
- 配置 Tailwind 深色户外主题色板
- 搭建全局 Layout（顶部导航 + 底部 Tab）

**P1-2: 首页 — 钓鱼指数仪表盘**

- 钓鱼指数大圆环（0-100 分，渐变色）
- 气象信息卡片（气温/气压/风向/湿度，Mock 数据）
- 一句话鱼情总结
- 快捷入口（开始推荐 / 查看水层 / 提交反馈）

**P1-3: 推荐页 — 条件输入 + 结果展示**

- 顶部条件输入面板：地点输入框（暂不接地图）、水域类型选择器、目标鱼种选择器（可选）
- 鱼种推荐卡片列表：鱼名 + 推荐星级 + 水层标签 + 活跃时段
- 展开详情：钓法/饵料/线组/技巧
- 时间轴图表：24h 活跃度曲线（Recharts）
- 环境分析栏：气压评分 + 风向建议 + 注意事项

**P1-4: 水层剖面图页**

- SVG 绘制水域横截面（从水面到底层，渐变蓝色）
- 各水层标注鱼种图标 + 名称 + 出现概率
- 鱼种水层偏移说明（气温/气压影响）

**P1-5: 反馈页 + 用户画像页**

- 渔获反馈表单：日期、钓点、鱼种、数量、钓法、满意度
- 用户画像页：标签气泡图 + 各标签成熟度进度条 + 置信度等级

### Phase 2: 接入后端功能（确认 UI 后）

**P2-1**: 接入和风天气 API → 首页气象数据实时化
**P2-2**: 接入高德地图 JS API → 地图选点 + POI 搜索
**P2-3**: 接入 DeepSeek API → 综合推荐功能（模块 B+C）
**P2-4**: 实现用户画像打标系统（localStorage 存储）
**P2-5**: 接入反馈流程（模块 E）+ 画像更新

### Phase 3: 优化与完善

- PWA 支持（离线缓存）
- 推荐结果缓存（相同条件短时间内不重复调用 LLM）
- 性能优化（流式输出、骨架屏）
- 响应式适配（PC 宽屏布局）
