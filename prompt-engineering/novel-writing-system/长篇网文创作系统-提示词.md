## v4.0 核心流程

```
第一阶段：基建锁定
  世界观 → 角色 → 关系 → 故事概览卡 → 整体评分≥95 → 用户预览确认 → 🔒锁定

第二阶段：开篇预览
  详细大纲 → 大纲评分≥90 → 黄金三章 → 开篇评分≥90 → 用户预览确认 → 📌定稿

第三阶段：连载生产
  第4章起循环：写章 → 状态更新 → 连贯性检查 → 爽感审计 → 章节评分≥80
```
