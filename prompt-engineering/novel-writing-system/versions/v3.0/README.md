# 长篇网文创作系统 v3.0 — 提示词模板

> 一套模块化的提示词系统，用于驱动 LLM 自动生成并持续创作长篇网文小说。
> v3.0 结合番茄小说、起点中文网、书旗小说、七猫小说等平台的爆款规律全面升级。

---

## v3.0 升级亮点

| 升级项 | 说明 |
|--------|------|
| **平台适配策略** | 针对番茄/起点/书旗/七猫/知乎盐选等平台的算法和读者特征，定制差异化写作参数 |
| **字数分段体系** | 短篇(5-20万) / 中篇(20-60万) / 长篇(60-150万) / 超长篇(150万+) 四档独立策略 |
| **黄金三章模块** | 独立的开篇设计系统，首章完读率>15%、追读率>30% 的工业级标准 |
| **8大类型引擎** | 在 v2.0 的4个引擎基础上，新增言情、悬疑推理、都市、历史架空4个引擎 |
| **爽感审计模块** | 独立的节奏质量检测工具，量化评估每章的爽感执行情况 |
| **跨类型融合指南** | 支持"玄幻+系统""言情+悬疑""都市+重生"等复合类型的融合写作 |
| **全流程质量评分** | 4级评分门禁：基建≥95 / 大纲≥90 / 开篇≥90 / 章节≥80，不达标回炉重写 |

## 使用流程

### 第一步：确定基础参数

填写以下信息：

```yaml
小说类型: ""          # 玄幻/科幻/言情/悬疑/都市/历史/规则怪谈/系统流（可组合）
目标平台: ""          # 番茄/起点/书旗/七猫/知乎盐选/通用
目标总字数: ""        # 系统会自动匹配字数分段策略
每章字数: ""          # 留空则根据平台自动推荐
故事基调: ""          # 热血爽文/暗黑虐心/轻松搞笑/悬疑烧脑/甜宠治愈
核心卖点: ""          # 系统流/重生/穿越/规则怪谈/双强/甜宠/种田...
性别频道: ""          # 男频/女频/双性向
```

### 第二步：按顺序执行（每步都有评分门禁）

```
1. 阅读 [00-平台适配与字数分段](./00-平台适配与字数分段.md)
   → 确认平台策略和字数档位

2. 阅读 [01-类型写作引擎](./01-类型写作引擎.md)
   → 确认激活哪些引擎（可多选组合）

3. 执行 [03-五阶段核心提示词](./03-五阶段核心提示词.md) 的阶段一~三
   → 生成世界观→角色→关系
   → 每个产出用 [05-质量评分体系](./05-质量评分体系.md) 的【基建评分】评分，≥95分才继续

4. 执行阶段四（章节大纲）
   → 用【大纲评分】评分，≥90分才继续

5. 执行 [02-黄金三章与开篇设计](./02-黄金三章与开篇设计.md)
   → 生成前3章正文
   → 用【开篇评分】评分，≥90分才继续（不达标回炉重写）

6. 执行阶段五（第4章起循环写作）
   → 每章写完后执行 [04-辅助模块](./04-辅助模块.md) + [05-质量评分体系](./05-质量评分体系.md)【章节评分】
   → ≥80分继续，<80分修改
```

### 循环写作流程

```
写第N章 → 状态更新器 → 连贯性检查 → 爽感审计 → 章节评分 ≥80? → 写第N+1章
                                                    ↑         │
                                                    └── 否 ── 修改后重评
```

## 快速启动（极简版）

如果你想快速开始，只需要：

1. 填好上面的基础参数
2. 直接跳到 [03-五阶段核心提示词](./03-五阶段核心提示词.md) 的阶段一
3. 按阶段一→二→三→四→五的顺序执行
4. 每章写完后用 [04-辅助模块](./04-辅助模块.md) 做检查

黄金三章模块和类型引擎会在阶段提示词中自动引用，不需要单独执行。
